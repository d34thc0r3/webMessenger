# 🗨️ WebMessenger

수업시간에 옆자리 친구랑 **메모장에 글자 적으면서 대화**하다가  
“컴퓨터공학과 학생이 이게 맞나…?” 하는 **자괴감**이 들어서 만들었습니다.

그래서 만든 게 이겁니다.

---

## ✨ 소개

**WebMessenger**는  
- 설치 ❌  
- 회원가입 ❌  
- 앱 ❌  

그냥 **주소창에 입력 → 바로 대화**하는  
초경량 웹 기반 메신저입니다.

IRC 감성의 순수 텍스트 채팅을 목표로 했습니다.

---

## 🚀 사용 방법

1. 웹 주소로 접속
2. 방 번호 입력
3. 닉네임 입력
4. 끝. 바로 대화 시작

> 같은 방 번호를 입력한 사람들끼리 실시간으로 채팅할 수 있습니다.

---

## 🧩 특징

- 📦 설치 필요 없음 (브라우저만 있으면 OK)
- 🔢 방 번호 기반 채팅
- ⚡ WebSocket 기반 실시간 통신
- 🖥️ 순수 텍스트 채팅 (메모장/IRC 느낌)
- 🕶️ 채팅 기록 저장 안 함 (서버 메모리 기반)
- 대화한 친구 고소할꺼면 스크린샷 찍으셔요

---

## 🛠️ 기술 스택

- **Backend**
  - Node.js
  - Express
  - Socket.IO

- **Frontend**
  - HTML / Vanilla JS

- **Deploy**
  - Render (Free Plan)

---

## ⚠️ 참고 사항

- 무료 플랜을 사용 중이라  
  일정 시간 사용하지 않으면 서버가 잠들 수 있습니다.
- 처음 접속 시 약간 느릴 수 있으나,  
  한 번 연결되면 정상적으로 작동합니다.
